## Buying Guide

### What to Consider

1. **Purpose**: What will you primarily use the device for?
2. **Budget**: Set a realistic price range
3. **Specifications**: Match specs to your needs
4. **Compatibility**: Ensure compatibility with existing devices
5. **Warranty**: Check warranty coverage and terms

### Understanding Specifications

#### For Laptops
- **Processor (CPU)**: Intel Core i5/i7 or AMD Ryzen for everyday use
- **RAM**: 8GB minimum, 16GB+ for heavy multitasking
- **Storage**: SSD preferred, 256GB minimum
- **Display**: 1080p for basic use, 4K for content creation

#### For Smartphones
- **Display**: OLED for best colors, 120Hz for smooth scrolling
- **Camera**: 12MP+ for good photos, consider optical zoom
- **Battery**: 4000mAh+ for all-day use
- **Storage**: 128GB minimum, consider cloud options

## Care & Maintenance

### General Tips
1. Keep devices clean and dust-free
2. Avoid extreme temperatures
3. Use surge protectors
4. Update software regularly
5. Back up your data

### Battery Care
- Avoid letting battery drain to 0%
- Don't leave plugged in at 100% for extended periods
- Use original or certified chargers
- Keep devices cool during charging

### Screen Care
- Use microfiber cloth for cleaning
- Apply screen protector for mobile devices
- Avoid direct sunlight for extended periods
- Adjust brightness to reduce strain

## Troubleshooting

### Common Issues

#### Device Won't Turn On
1. Check if battery is charged
2. Try a different power outlet/cable
3. Perform a hard reset (hold power 10+ seconds)
4. Contact support if issue persists

#### Slow Performance
1. Close unused applications
2. Clear cache and temporary files
3. Check available storage space
4. Restart the device
5. Check for software updates

#### Connectivity Issues
1. Toggle Wi-Fi/Bluetooth off and on
2. Forget and reconnect to network
3. Restart router/modem
4. Check for interference sources
